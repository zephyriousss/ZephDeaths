# ZephDeaths

Per-player death tracking for WoW Mythic+ dungeons (Retail 12.0+).

Replaces the default Blizzard death count skull with a movable, customizable widget showing total deaths on the icon and a per-player breakdown with class colors on hover.

## Features
- Per-player death count with class-colored tooltip
- Time lost display during active keystones
- Movable, lockable, screen-clamped
- Customizable icon size, font size, font color, text position
- Option to hide the default Blizzard death counter
- Death flash effect on the skull icon
- Manual tracking and testing modes for use outside keystones (so you can configure it before entering a key)


## Install
Download the two files and place them in:
`World of Warcraft/_retail_/Interface/AddOns/ZephDeaths/`

## Commands
- `/zd` — show help
- `/zd options` — open settings
- `/zd show` / `hide` — toggle tracker
- `/zd lock` / `unlock` — lock position
- `/zd reset` — reset position
- `/zd track` / `stop` — manual tracking mode
- `/zd test` — load test data
